# 🚀 Smart Resume AI

An AI-powered web application that analyzes resumes, provides detailed feedback, and evaluates how well a resume matches a specific job role.

---

## 📌 Features

* 📄 Resume analysis with structured feedback
* 🎯 Job-role compatibility scoring
* 🧠 Keyword and skill extraction
* 📊 Suggestions to improve resume quality
* ⚡ Interactive and user-friendly interface

---

## 🛠️ Tech Stack

**Frontend:** JavaScript, HTML, CSS
**Backend:** Python
**Database:** SQL
**Other Tools:** Git, GitHub

---

## 💡 How It Works

1. User uploads their resume
2. System analyzes content (skills, experience, keywords)
3. Compares resume with selected job role
4. Generates:

   * Compatibility score
   * Improvement suggestions
   * Missing keywords

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/Resume-AI.git
cd Resume-AI
```

### 2. Setup Backend

```
cd backend
pip install -r requirements.txt
python app.py
```

### 3. Setup Frontend

```
cd frontend
npm install
npm start
```
